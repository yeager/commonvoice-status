# Common Voice Status

View Mozilla Common Voice recording statistics per language.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install commonvoice-status
```

### Fedora/RPM
```bash
sudo dnf install commonvoice-status
```
